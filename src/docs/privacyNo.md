# Personvernerklæring

## 1. Innledning

Denne personvernerklæringen forklarer hvilke data Utvekslingsportalen samler inn, hvordan dataene brukes, og hvilke rettigheter du har som bruker.

Utvekslingsportalen er et studentdrevet, ikke-kommersielt prosjekt og er **ikke en offisiell NTNU-tjeneste**.

## 2. Hvilke data vi samler inn

### 2.1 Brukerkontoer og autentisering

Når du oppretter en konto eller logger inn, kan vi samle inn:

- E-postadresse
- Kontoidentifikator levert av autentiseringsmetoden

Autentisering håndteres ved bruk av **Firebase Authentication**, inkludert:

- Innlogging med e-post og passord
- Innlogging med Google

Vi har **ikke tilgang til ditt Google-passord** eller private data fra Google-kontoen din.

### 2.2 Brukerinnsendt innhold

Dersom du sender inn en utvekslingserfaring, kan vi lagre:

- Universitet, land, år/semester
- Studieprogram og eventuell studieretning
- Faginformasjon (emnekoder, emnenavn, studiepoeng, erstattede emner)
- Kommentarer du selv velger å dele

Ikke send inn sensitive personopplysninger eller unødvendig personlig informasjon.

### 2.3 Analyseverktøy

Utvekslingsportalen bruker **Google Analytics (GA4)** i produksjon for å forstå hvordan plattformen brukes og for å forbedre funksjonalitet og brukeropplevelse.

Analyse­data:

- Samles inn i aggregert form
- Brukes til bruksstatistikk (for eksempel sidevisninger og navigasjonsmønstre)
- Brukes ikke til annonsering eller profilering

## 3. Hvordan vi bruker data

Innsamlede data brukes til å:

- Autentisere brukere
- Vise utvekslingserfaringer til andre studenter
- Forbedre plattformen og brukeropplevelsen
- Ivareta sikkerhet og forhindre misbruk

## 4. Lagring og behandling

Data lagres ved bruk av **Firebase** på Google Cloud-infrastruktur. Det benyttes rimelige tekniske og organisatoriske tiltak for å beskytte lagrede data.

## 5. Deling av data

- Personopplysninger **selges ikke**
- Data **deles ikke til markedsføring eller annonsering**
- Analyse­data behandles av Google i henhold til deres standardvilkår

## 6. Lagringstid

Kontodata og brukerinnsendt innhold lagres så lenge det er nødvendig for å drifte plattformen, eller frem til sletting blir forespurt.

## 7. Dine rettigheter

Du har rett til å:

- Be om innsyn i data som er lagret om deg
- Be om retting av uriktige opplysninger
- Be om sletting av konto eller brukerinnsendt innhold
