# Ansvarsfraskrivelse

Utvekslingsportalen er et **studentinitiert, ikke-kommersielt prosjekt** og er **ikke en offisiell NTNU-tjeneste**.

All informasjon gis kun for **informasjonsformål** og kan være ufullstendig eller utdatert.

Du er selv ansvarlig for å verifisere:

- Utvekslingsavtaler og regler
- Fag- og kursgodkjenninger
- Krav i studieprogrammet

Bekreft alltid informasjonen med NTNU og den aktuelle partnerinstitusjonen.
